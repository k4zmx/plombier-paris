# Docteur Plomberie — Landing Page

A fast, SEO-optimized landing page for a French plumbing emergency service, built with Astro.

## Tech Stack
- [Astro](https://astro.build/) — static site generator
- TypeScript
- Vanilla CSS

## Features
- Emergency plumbing service landing page targeting French market
- Optimized for Core Web Vitals and SEO
- Mobile responsive
- Fast load times with zero JS by default

## Project Structure
- `src/` — components and pages
- `public/` — static assets
- `dist/` — production build output
